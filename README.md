Controls:
WASD - Move left/right/forward/back
Q - Descend
E - Ascend
Z - Rotate left
X - Rotate right
R - Reset camera
Esc - Quit
1 - Default Mode
2 - Debug Mode
3 - Post Processing Mode
4 - Performance Mode
5 - Fog Mode
6 - Wireframe Mode
